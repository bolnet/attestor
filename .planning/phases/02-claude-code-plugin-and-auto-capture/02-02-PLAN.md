Claude Code hook protocol (stdin/stdout JSON):
- Hooks receive JSON on stdin with event-specific payload
- SessionStart: `{"event": "SessionStart", "session_id": "...", "cwd": "..."}`
- PostToolUse: `{"event": "PostToolUse", "session_id": "...", "tool": {"name": "Write", "input": {...}, "output": "..."}, "cwd": "..."}`
- Stop: `{"event": "Stop", "session_id": "...", "cwd": "..."}`
- Hooks return JSON on stdout
- SessionStart returns: `{"additionalContext": "string to inject"}`
- PostToolUse returns: `{}` (side-effect only)
- Stop returns: `{}` (side-effect only)

- Also has `def main()` that reads stdin JSON, calls handle(), writes stdout JSON
