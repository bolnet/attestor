The Problem

AI agents forget everything between conversations. Cloud memory services add latency, vendor lock-in, and zero visibility. Most require Docker, API keys, and complex setup before you write a line of code.

The Solution

Memwright runs entirely on your machine with zero configuration. pip install memwright gives you tag matching, entity graph traversal, and semantic vector search — all fused with Reciprocal Rank Fusion. No Docker. No API keys. Your agent's memory is a directory you can inspect, query, and version control.

The Stack

🗃

SQLite

Core storage. ACID guarantees. Always available. Run raw SQL to inspect your memories. Built into Python — zero dependencies.

🔭

ChromaDB

In-process vector store with built-in sentence-transformers embeddings (all-MiniLM-L6-v2). No API keys. Persistent on disk.

🔗

NetworkX

In-memory entity graph with multi-hop BFS traversal. JSON persistence. Sub-millisecond queries. Entities and relationships extracted automatically.

📈

RRF Fusion

Reciprocal Rank Fusion combines results from all three layers. Memories appearing in multiple layers get boosted. Temporal and entity boosts applied.

Quick Start

Claude Code Plugin (recommended)
# Install plugin — auto-captures memories
/plugin marketplace add bolnet/agent-memory

# That's it. Memories captured automatically.
# SessionStart injects context.
# PostToolUse captures observations.
# Stop summarizes decisions.
MCP Server (any client)
# 1. Install
pip install memwright

# 2. Add to .mcp.json
{
  "mcpServers": {
    "memory": {
      "command": "memwright",
      "args": ["mcp"]
    }
  }
}

# Works with Claude Code, Cursor,
# Windsurf, or any MCP client.
Python API
from agent_memory import AgentMemory

mem = AgentMemory("./my-agent")  # auto-provisions SQLite + ChromaDB + NetworkX

# Store facts
mem.add("User prefers Python over Java", tags=["preference"], category="preference")

# Recall relevant memories (tag + graph + vector fusion)
results = mem.recall("what language?")

# Get formatted context for prompt injection
context = mem.recall_as_context("user background", budget=20000)
